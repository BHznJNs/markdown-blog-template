# Independent Page Embedded

You can embed another webpages by their links:
```markdown
@[External Link](https://www.w3.org/)
@[Internal Link](.resources/test.html)
```
@[External Link](https://www.w3.org/)
@[Internal Link](.resources/test.html)

You can also write HTML code in markdown file:
```markdown
@@@
<h1>Test</h1>
@@@
```
@@@
<h1>Test</h1>
@@@
